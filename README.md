## Funcionalidades
- Autenticação (email e senha)
- Detalhes do usuário
- Cadastro de registro
- Listagem de registros
- Deletar registro
- Filtro por data

## Stack utilizada
- React Native
- Expo
- Biome
- Husky
- Typescript
- React Navigation
- axios
- date fns
